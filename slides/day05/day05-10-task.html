Day 5 · 下午任务

任务布置

下午任务：冲刺项目核心功能与整合

时间：14:00 - 17:30（3.5 小时，含 15 分钟休息）| 目标：完成以下三项任务

1

Trae 核心模块完成

  • 项目页面有完整的界面（导航栏、内容区、页脚）
  • 页面能正确读取 WorkBuddy 处理后的数据文件
  • 数据以表格或卡片形式展示，无 undefined 或空白
  • 至少实现一个交互功能（如搜索、筛选、排序）
  • 如果条件允许，接入一个公开 API 显示实时数据

验收标准：在浏览器中打开页面，能看到真实数据，交互正常

2

三工具整合联调

  • QClaw 采集的数据能自动保存到约定位置
  • WorkBuddy 能读取 QClaw 输出并正确清洗
  • Trae 页面能读取 WorkBuddy 的输出并展示
  • 全流程跑通一次：采集 → 清洗 → 展示，无需手动复制粘贴
  • 数据文件缺失时，页面显示友好提示

验收标准：修改 QClaw 数据后，刷新 Trae 页面能看到更新

3

项目文档初稿

  • README.md：项目简介、功能特性、安装步骤、团队成员
  • 技术方案文档：数据流说明、模块分工、接口约定
  • 使用说明：如何运行项目、常见问题
  • 文档用 Markdown 格式，保存在项目 docs/ 目录

验收标准：陌生人阅读 README 后，能独立运行项目
