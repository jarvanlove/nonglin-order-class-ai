进阶挑战

任务拓展：更多数据源 + 页面美化

如何添加更多 API 数据源

完成基础功能后，可以尝试接入更多 API，让项目数据更丰富。在 Trae Chat 中说："帮我再添加一个 XX API 的调用"，AI 会自动修改代码。

天气 显示多城市天气对比
新闻 农业/林业热点新闻聚合
汇率 农产品进出口汇率换算
地图 在地图上标注农业产区
日历 显示农事节气提醒

拓展注意事项

1. 每添加一个 API，都要测试其在网络不稳定时的表现

2. 注意 API 调用频率限制，不要频繁刷新

3. 如果 API 需要 Key，提前确认是否免费

4. 优先完成基础功能，拓展是"锦上添花"

页面美化技巧（不用写 CSS）

所有美化需求都可以用自然语言描述给 Trae，让 AI 生成 CSS。以下是常用的美化方向：

配色方案

"把页面改成绿色主题，适合农业项目"

卡片样式

"把列表改成卡片形式，带阴影和圆角"

响应式布局

"让页面在手机上也正常显示"

动画效果

"给卡片添加悬停放大效果"

图标美化

"在天气状况旁添加对应的图标"

表格优化

"给表格添加斑马纹和悬停高亮"

Trae 美化提示词模板

"请美化这个页面：1. 使用简洁专业的配色 2. 添加卡片阴影和圆角 3. 表格添加悬停效果 4. 整体风格适合学术/农业类项目。不要改变功能逻辑。"

时间分配建议

如果基础功能已完成（14:00-16:00）：16:00-17:00 做拓展，17:00-17:30 写文档。

如果基础功能未完成：放弃拓展，专注完成核心功能 + 文档。
