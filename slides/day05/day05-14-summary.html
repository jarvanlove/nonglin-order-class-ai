Trae 进阶

Chat 模式适合局部修改，Builder 模式适合从零搭建。两种模式灵活切换，效率最高。

API 入门

API 是软件间的对话方式。Trae 可以自动生成分调用代码，你只需描述需求。

调试三板斧

从下往上读报错、检查文件路径、确认数据格式匹配。遇到报错不慌，系统排查。

三工具整合

QClaw 采集 → WorkBuddy 清洗 → Trae 展示。关键是字段名统一和数据格式一致。
