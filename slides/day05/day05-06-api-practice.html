1

选择一个公开 API

推荐从农业气象 API 开始，数据简单、返回稳定。在 Trae Chat 中问："推荐一个免费的农业气象 API"。

2

在 Trae 中调用 API

在 Builder 模式描述："创建一个网页，调用农业气象 API 显示当前区域的温度、湿度和农事建议"。

3

数据显示到页面上

AI 会自动生成 fetch 代码和 HTML 展示逻辑。确认 Plan 后执行，在浏览器中预览效果。

// Trae 自动生成的 API 调用代码示例

fetch('https://api.agriweather.example/v1/current')

.then(response => response.json())

.then(data => {

// 把数据插入到页面元素中

document.getElementById('region').textContent = data.region;

document.getElementById('temp').textContent = data.temp + '°C';

})

.catch(error => {

console.error('获取气象数据失败:', error);

});

实战演示：天气数据展示

目标：创建一个简单的气象卡片，显示区域、温度、湿度与农事建议。

Trae 提示词：

"帮我创建一个 HTML 页面，调用公开农业气象 API，显示当前区域的温度、湿度和农事建议。页面要有一个卡片样式，显示区域名、温度、湿度。如果 API 调用失败，显示友好的错误提示。"

联调技巧

如果 API 返回的数据结构和预期不同，把 API 返回的完整内容复制到 Trae Chat 中，说："API 返回了这个数据，帮我修改代码正确解析"。AI 会自动调整字段映射。

常见报错

CORS 错误：浏览器安全策略阻止跨域请求。解决：使用 Trae 的 Live Server 运行，或让 AI 改用代理方案。

404 错误：API 地址拼写错误或服务已下线。解决：检查 URL，或换一个 API。
