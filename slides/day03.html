上午实战

用 Builder 模式做个人主页

任务要求

一个单页面网页（HTML 文件）

包含：姓名、专业、个人简介、头像位、联系方式

风格：简洁、现代、浅色背景、响应式

技术：HTML + CSS（AI 自动生成）

你的操作流程

1. 打开 Trae，点击 Builder 按钮

2. 用自然语言描述需求（越具体越好）

3. 审阅 AI 生成的 plan，确认执行

4. 查看代码，替换占位符为真实信息

5. 在浏览器中预览效果

提示词示例
请帮我创建一个个人介绍网页，要求如下：
1. 页面标题为"我的个人主页"
2. 包含一个头像占位区域（圆形，使用占位图）
3. 显示姓名（用占位符"[你的姓名]"）
4. 显示专业/班级信息（占位符"[你的专业]"）
5. 一段个人简介（占位符"[你的简介]"，限制 100 字以内）
6. 联系方式区域：邮箱、电话（都用占位符）
7. 整体风格：浅灰色背景，白色内容卡片，简洁现代
8. 页面要居中显示，在手机上也能正常看（响应式）
9. 请生成一个完整的 HTML 文件，包含内联 CSS

项目骨架

推荐项目目录结构

project-name/ # 项目根目录
├── README.md # 项目说明文档（必须）
├── docs/ # 文档目录
│ ├── requirement.md # 需求文档
│ ├── tech-design.md # 技术方案
│ └── 分工表.md # 分工与进度
├── data/ # 数据目录
│ ├── raw/ # QClaw 原始数据
│ └── processed/ # WorkBuddy 处理后
├── src/ # 源代码（Trae 生成）
│ ├── index.html
│ ├── css/
│ └── js/
├── output/ # 输出目录
│ └── reports/ # WorkBuddy 报告
└── assets/ # 静态资源
└── images/
创建方式

1. 在 Trae 中打开空文件夹作为项目根目录

2. 文件树中右键 → "新建文件夹"

3. 或用 Builder 模式说：

"帮我创建标准的项目目录结构"
