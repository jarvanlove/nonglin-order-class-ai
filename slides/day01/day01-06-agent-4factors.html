核心概念

Agent 四要素：什么让 Agent 不同于聊天工具？

1

自主性

Agent 能自己决定下一步做什么，不需要人类逐步指令。你给一个大目标，它会自动拆解成子任务并依次执行。

举例

你说"帮我订一张明天去北京的机票"，Agent 会自动：查航班 → 比价 → 选座位 → 填信息 → 下单

2

记忆性

Agent 能记住上下文和历史偏好，越用越懂你。不同于聊天工具的"金鱼记忆"，Agent 有长期记忆能力。

举例

你上周说"我喜欢靠窗座位、不要红眼航班"，这周 Agent 会自动按这个偏好帮你筛选

3

工具调用

Agent 能使用搜索、文件处理、发邮件、操作数据库等外部工具，像人类使用软件一样扩展自己的能力边界。

举例

需要查天气时调用天气 API，需要发报告时调用邮件服务，需要整理数据时操作 Excel

4

持续运行

Agent 不需要你每一步都盯着，能自主推进任务直到完成。你可以给它一个任务后去干别的，回来收结果。

举例

你说"监控这个网站的价格变化，降价到 500 以下就通知我"，Agent 会持续监控，不需要你一直问
