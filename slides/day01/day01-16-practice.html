安装常见问题

下载速度很慢或失败？

尝试手机热点下载，或向已下载的同学拷贝安装包。

微信扫码后无法绑定？

确保微信是最新版本，退出重登后再扫码。仍失败请联系助教。

提示"系统版本过低"？

Windows 7 / macOS 11 及以下不支持。请联系讲师获取云端替代方案。

安装后无法连接网络？

确认未开启 VPN，尝试切换校园网有线/无线连接。

Prompt 写法要点

  • 明确时间范围（"近 3 年"而非"最近"）
  • 指定分类维度（"按技术类型分类"）
  • 要求输出格式（"整理成表格"）
  • 设定数量限制（"至少 5 个案例"）
  • 补充背景信息（"面向农林高校学生"）

Prompt 示例

基础版

搜集 AI 在农业中的应用

进阶版（推荐）

搜集近 3 年（2023-2026）AI 在农林病虫害识别领域的应用案例，按"技术类型"和"应用场景"两个维度分类，每个案例包含：案例名称、技术原理、应用效果、来源链接。整理成 Markdown 表格。

高阶版

作为农林高校学生的课程作业，我需要一份关于"智慧农业"的研究资料。请搜集近 3 年国内外智慧农业的典型案例，重点覆盖：物联网监测、AI 病虫害识别、无人机植保、精准灌溉 4 个方向。每个案例 100 字左右，包含关键数据。最后附上一份推荐阅读清单。

输出格式要求

  • 文件名：姓名_农林AI案例_日期.txt / .md
  • 必须包含：案例名称、应用场景、技术类型、效果数据、来源
  • 建议用表格或结构化列表，方便后续使用
  • 保存到桌面，下午下课前提交给助教
