## 六、动画规范（从现有HTML提取）

### 6.1 入场动画类型

| 动画名 | 效果 | 时长 | 用途 |
|-------|------|------|------|
| fadeInUp | 从下淡入 | 0.5s | 主要内容 |
| fadeInDown | 从上淡入 | 0.5s | 顶部元素 |
| fadeInLeft | 从左淡入 | 0.5s | 左侧元素 |
| fadeInRight | 从右淡入 | 0.8s | 右侧装饰 |
| scaleIn | 缩放淡入 | 0.6s | 装饰圆点 |
| slideInUp | 上滑+缩放 | 0.55s | 统一内容入场 |

### 6.2 延迟规则（stagger）

- 第1个元素: 0s
- 第2个元素: 0.1s
- 第3个元素: 0.2s
- 第4个元素: 0.3s
- 以此类推，间隔0.06-0.1s

### 6.3 缓动函数

- 默认: ease-out
- 内容入场: cubic-bezier(0.22, 1, 0.36, 1)

## 八、设计原则总结

1. **简洁专业**: 以白色/浅蓝白为底，深蓝为主色，避免花哨
2. **层次清晰**: 通过字号、颜色、边框建立视觉层级
3. **统一规范**: 所有内容页共享相同的标题栏结构
4. **适度装饰**: 封面有图形装饰，内容页保持干净
5. **动画克制**: 入场动画简洁，不干扰内容阅读
