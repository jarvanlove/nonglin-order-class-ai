实战演示

Builder 模式演示：从一句话到农产品展示页

输入 Prompt

"创建一个'今日菜价'农产品信息展示网页，顶部有绿色标题栏显示'XX农贸市场今日菜价'，中间用卡片展示 4 种蔬菜（西红柿、黄瓜、茄子、菠菜），每张卡片包含蔬菜名称、产地、价格（元/斤）和涨跌箭头，绿色表示下跌、红色表示上涨，底部有灰色页脚显示'价格仅供参考'。"

Trae 生成的代码

<!DOCTYPE html>
<html>
<head>
  <style>
    .header { background: #4a8a2a; color: #fff;
      padding: 16px; text-align: center; }
    .card-grid { display: flex; gap: 12px;
      padding: 20px; justify-content: center; }
    .card { border: 1px solid #e2e8f0;
      border-radius: 8px; padding: 14px;
      width: 140px; text-align: center; }
    .price { font-size: 24px; color: #084E8F;
      font-weight: 700; }
    .up { color: #92181A; }
    .down { color: #4a8a2a; }
    .footer { background: #f5f5f5; color: #666;
      text-align: center; padding: 10px; }
  </style>
</head>
<body>
  <div class="header">XX农贸市场今日菜价</div>
  <div class="card-grid">
    <div class="card"><h3>西红柿</h3>
    <p>产地：寿光</p>
    <p class="price down">2.8元/斤 ↓</p>
    </div>
    ...
  </div>
  <div class="footer">价格仅供参考</div>
</body>
</html>
