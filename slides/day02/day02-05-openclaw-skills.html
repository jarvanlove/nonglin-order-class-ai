OpenClaw Skills：让 Agent 从"会聊天"到"会做事"

Skills 是什么？

Skills 是 OpenClaw 的"功能插件"，类似于手机上的 App。没有 Skills 的 Agent 只能聊天；安装了 Skills 的 Agent 可以搜索网页、处理文件、发送微信、查询天气……Skills 让 Agent 的能力无限扩展。

🔍

搜索 Skill

调用搜索引擎，实时获取互联网信息，支持多轮搜索和结果汇总

📄

文件处理 Skill

读取、写入、解析各类文件（Excel、Word、PDF、CSV）

💬

微信推送 Skill

绑定微信后，Agent 可以主动发消息、推送报告、提醒任务

🌎

天气查询 Skill

获取实时天气、未来预报，支持多城市对比

📈

数据分析 Skill

对数据进行统计计算、生成图表、趋势分析

💻

代码执行 Skill

运行 Python、Shell 等代码，实现自定义逻辑

Skills 开发示例

开发一个 Skill 非常简单，只需要定义：名称、描述、参数、执行逻辑。

# Skill 定义示例：天气查询

name: weather_query

description: 查询指定城市的天气

params:

  - city: 城市名称（如"北京"）

logic:

  1. 调用天气 API

  2. 解析返回数据

  3. 格式化输出

关键洞察

OpenClaw 的社区已贡献数千个 Skills，覆盖办公、开发、生活、教育等各个领域。你不需要自己开发所有 Skill，而是像逛应用商店一样，找到需要的直接安装使用。这也是 OpenClaw 能获得 33 万 Stars 的核心原因——开放生态。
