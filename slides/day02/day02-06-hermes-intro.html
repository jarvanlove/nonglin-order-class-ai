Hermes Agent：会自我进化的智能体

Hermes Agent 是什么？

Hermes 是另一个前沿开源 Agent 项目，与 OpenClaw 的最大区别在于：它能自我进化。OpenClaw 像一个功能丰富的工具箱——你有什么需求，就装什么 Skill；Hermes 像一个不断学习的实习生——它会在工作中总结经验，越做越好。

"自我进化"是什么意思？

1

第一次使用

你让 Hermes 整理一份实验报告，它用了一种格式，你反馈"表格太宽了"

2

记录反馈

Hermes 记住"这个用户喜欢窄表格"，调整提示词策略

3

第二次使用

同样任务，Hermes 自动使用窄表格格式，无需你再次提醒

4

持续优化

使用次数越多，Hermes 越懂你的偏好，输出质量持续提升

Hermes 本地部署配置

最低配置：4核 CPU / 8GB 内存 / 20GB 硬盘（无 GPU 可用 CPU 推理）
推荐配置：8核 CPU / 16GB 内存 / NVIDIA RTX 3060 8GB+ / 50GB SSD
提示：Hermes 比 OpenClaw 更轻量，7B 模型可在 8GB 内存设备流畅运行。自我进化产生的策略缓存约占用 500MB~2GB 空间。

OpenClaw 是"给你工具，你自己搭"；Hermes 是"你只管用，我自己学"。两者互补——OpenClaw 解决"能做什么"，Hermes 解决"做得好不好"。
