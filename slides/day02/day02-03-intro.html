Day 1 我们做了什么？

用 QClaw 完成了信息搜集和资料整理——这是 Agent 的"基础用法"。今天我们进入"进阶用法"：了解 Agent 的底层架构，以及如何让 Agent 自己变得更聪明、更懂你。

核心问题

如果让 Agent 自己变聪明，会发生什么？

思考一分钟：昨天你让 QClaw 搜集资料，它执行了你的指令。但如果它执行完后自己"复盘"——发现某种搜索方式效果更好，下次自动调整——这会带来什么变化？

这就是今天我们要探索的：Agent 的自我进化。不是人教 Agent 怎么做，而是 Agent 自己学会怎么做更好。
