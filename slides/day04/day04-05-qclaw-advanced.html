QClaw 进阶

Skills 调用、格式控制与批量处理

Skills 调用进阶

在 prompt 中显式声明需要调用的 Skill，并指定参数，可大幅提升输出质量。

请使用 [WebSearch] 搜索"2024 年小麦病虫害

AI 识别准确率"，然后用 [DataExtract]

提取：病害名称、识别技术、准确率、来源。

输出为 JSON 数组格式。

技巧：先声明 Skill，再描述任务，最后指定格式——三步结构最稳定。

输出格式控制

根据下游工具的需求，精确控制 QClaw 的输出格式：

格式
适用场景
Prompt 写法
JSON
程序读取、结构化数据
"输出标准 JSON，不要 markdown 代码块"
CSV
Excel 导入、表格处理
"输出 CSV 格式，第一行为表头"
Markdown
报告、文档、展示
"输出 Markdown 表格，带标题"

批量处理技巧

  • 批量 URL 采集：将多个链接放入一个 prompt，要求 QClaw 逐条处理并统一输出
  • 分页处理：对长列表数据，要求分批次输出，每批附带序号便于合并
  • 模板化 Prompt：将常用任务写成模板，只替换变量部分，保持输出格式一致
  • 增量更新：先输出已有数据摘要，再要求"基于以上，补充新采集的数据"
  • 错误重试：某条数据处理失败时，单独提取失败项重新提交，不影响已完成的

进阶 Prompt 模板示例

任务：采集 {目标网站} 上关于 {主题} 的信息

要求：

1. 使用 [WebSearch] 搜索相关页面

2. 使用 [DataExtract] 提取关键字段

3. 输出 JSON 数组，每个对象包含：

title, source, date, summary, url

4. 确保所有字段不为空，日期统一为 YYYY-MM-DD

将 {目标网站} 和 {主题} 替换为实际值即可复用，保证输出结构一致。
