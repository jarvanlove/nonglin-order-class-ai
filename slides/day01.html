范式转移
从 Chat 到 Agent
?

Chat 模式

你请了一个非常聪明的顾问，坐在你对面。你问他什么，他答什么。
−不会主动做事
−不记住历史对话
−不调用外部工具
−需要你一直盯着
本质：会说话的知识库
→
A

Agent 模式

你请了一个数字员工。交代目标，它自主执行，主动通知你结果。
+自主规划并执行步骤
+记住上下文与偏好
+调用搜索、微信、Excel
+后台运行，完成通知
本质：有大脑 + 有手脚的人

Agent 四要素
1
Autonomy

自主性

给定目标后，能自主规划步骤并执行，不需要人类每一步指令

2
Memory

记忆性

记住上下文、用户偏好、历史任务，越用越懂你

3
Tool Use

工具调用

调用搜索引擎、数据库、微信、Excel、代码执行环境等外部工具

4
Persistence

持续运行

不需要你一直盯着，可以在后台持续工作，完成后主动通知你
