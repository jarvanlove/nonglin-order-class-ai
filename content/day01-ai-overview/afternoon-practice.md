## 一、Agent 概念入门讲解（14:00 - 14:30，30 分钟）

### 1.1 从"聊天工具"到"数字员工"

> **【讲师话术】**
> 上午我们讲了 Agent 的四要素：自主性、记忆性、工具调用、持续运行。下午开始动手之前，我再用一个具体的例子帮大家建立直觉。
>
> 假设你是一个农业技术推广站的实习生，领导让你做一件事："了解一下近 3 年 AI 在病虫害识别方面的应用，整理一份资料，下班前发我。"
>
> **传统方式**：你打开百度，搜"AI 病虫害识别"，点开十几个网页，复制粘贴到 Word，自己分类、排版，花 3-4 小时做完。
>
> **Chat 方式**：你打开 ChatGPT，问"AI 在病虫害识别方面有哪些应用？"它给你一段文字。但这段文字可能不够新、不够全、没有出处，而且你不能直接让它去搜、去整理、去发微信。
>
> **Agent 方式**：你对 QClaw 说："搜集 2023-2026 年 AI 在农林病虫害识别领域的应用案例，按'技术类型、应用场景、效果数据、来源链接'整理成表格，完成后发到我微信。"然后 QClaw 自己去搜索、去筛选、去格式化，最后直接把整理好的结果推送到你的微信。你可以去干别的事，等它通知你。
>
> 这就是"数字员工"和"聊天工具"的本质区别。下午你们就要体验这个过程。

### 1.2 QClaw 的定位与优势

| 特性 | 说明 |
|------|------|
| **微信直连** | 扫码即用，无需注册邮箱、无需科学上网、无需配置 API Key |
| **零门槛** | 非计算机专业学生 5 分钟内完成安装和绑定 |
| **免费额度充足** | 每天 4000 万 Token，正常使用完全用不完 |
| **基于 OpenClaw** | 底层是开源 Agent 平台，功能可扩展，Skills 生态丰富 |
| **国内网络可用** | 纯国内环境访问，符合学校网络管理要求 |

> **【讲师话术】**
> QClaw 是腾讯基于 OpenClaw 做的本地化适配。OpenClaw 是什么？就是上午讲的那个 GitHub 33 万 Star 的开源 Agent 平台。QClaw 继承了它的核心能力，但针对国内用户做了三件事：接入了微信、服务器放在国内、界面全中文。对你们来说，最大的好处就是"扫码即用"。

## 二、QClaw 安装与配置（14:30 - 15:30，60 分钟）

### 2.1 环境检查

在开始安装前，请确认以下条件：

1. **操作系统**：Windows 10/11、macOS 10.15+、或主流 Linux 发行版
2. **网络环境**：能正常访问国内网站（如 qq.com、baidu.com）
3. **微信**：手机微信已安装，能正常扫码
4. **磁盘空间**：至少 500MB 可用空间

> **【讲师话术】**
> 先检查一下你们的电脑。Windows 和 Mac 都没问题，Linux 也可以。关键是网络——只要能打开 QQ 官网，就能用 QClaw。手机微信准备好，等下要扫码。

### 2.2 下载 QClaw

**步骤 1：打开浏览器，访问官网**

1. 打开电脑上的浏览器（Chrome、Edge、Firefox、360 浏览器均可）
2. 在地址栏输入：`https://qclaw.qq.com`
3. 按回车键，等待页面加载

**预期界面描述**：
- 页面顶部：腾讯 QClaw 的 Logo 和导航栏（首页、下载、文档、社区）
- 页面中部：大幅宣传图，主标题通常为"QClaw - 微信远程办公 AI 助手"，副标题强调"零门槛、微信直连、每天 4000 万免费 Token"
- 页面下方：功能介绍区域，展示"信息搜集、文献整理、微信自动化"等核心能力
- 显眼位置有一个"立即下载"或"免费使用"按钮（通常为蓝色或绿色）

**步骤 2：点击下载按钮**

1. 找到页面上的"立即下载"按钮
2. 用鼠标左键单击该按钮
3. 浏览器会开始下载安装包，或跳转到下载页面

**预期界面描述**：
- 如果直接下载：浏览器底部或右上角出现下载进度条，文件名为 `QClaw-Setup-x.x.x.exe`（Windows）或 `QClaw-x.x.x.dmg`（Mac）
- 如果跳转到下载页面：页面显示多个版本选项（Windows、Mac、Linux），根据你的系统选择对应版本，然后点击下载

**步骤 3：等待下载完成**

- 文件大小约 100-200MB
- 根据网络速度，下载时间约 1-5 分钟
- 下载完成后，浏览器会提示"下载完成"或在下载列表中显示

> **【讲师话术】**
> 如果官网打不开，先检查你的网络能不能上 qq.com。如果 qq.com 能打开但 qclaw.qq.com 打不开，举手叫助教，可能是学校网络对特定域名有限制。我们准备了离线安装包作为备用。

### 2.3 安装 QClaw

**Windows 系统安装步骤**：

**步骤 4：运行安装程序**

1. 打开浏览器的下载列表，找到刚才下载的 `QClaw-Setup-x.x.x.exe`
2. 点击文件名，或右键选择"打开"
3. Windows 可能会弹出"用户账户控制"（UAC）提示，询问"是否允许此应用对你的设备进行更改？"
4. 点击"是"按钮

**预期界面描述**：
- 弹出 QClaw 安装向导窗口
- 窗口标题："QClaw 安装"
- 欢迎页面显示 QClaw Logo 和简介文字
- 底部有"下一步"按钮

**步骤 5：选择安装路径**

1. 点击"下一步"进入安装路径选择页面
2. 默认路径通常为 `C:\Users\你的用户名\AppData\Local\QClaw` 或 `C:\Program Files\QClaw`
3. **建议**：保持默认路径，不要修改（避免路径含中文或特殊字符导致问题）
4. 点击"下一步"

**步骤 6：完成安装**

1. 安装程序显示进度条，等待安装完成（约 1-2 分钟）
2. 安装完成后，页面显示"安装完成"
3. 勾选"运行 QClaw"（如果该选项存在）
4. 点击"完成"按钮

**预期界面描述**：
- QClaw 应用窗口自动打开
- 首次启动可能显示欢迎页或初始化进度

**macOS 系统安装步骤**：

1. 双击下载的 `.dmg` 文件
2. 在弹出的窗口中，将 QClaw 图标拖动到"应用程序"文件夹
3. 打开"启动台"，点击 QClaw 图标启动
4. 如果系统提示"无法打开，因为无法验证开发者"，前往"系统设置 -> 隐私与安全性"，点击"仍要打开"

**Linux 系统安装步骤**：

1. 下载 `.AppImage` 或 `.deb`/`.rpm` 包
2. 对于 `.AppImage`：右键文件 -> 属性 -> 勾选"允许作为程序执行"，然后双击运行
3. 对于 `.deb`（Ubuntu/Debian）：双击安装或使用命令 `sudo dpkg -i qclaw-x.x.x.deb`

### 2.4 微信扫码绑定

**步骤 7：获取绑定二维码**

1. QClaw 首次启动后，界面会显示一个二维码
2. 二维码旁边有文字提示："请使用微信扫一扫"
3. 二维码下方可能有"刷新二维码"按钮（如果二维码过期可点击）

**预期界面描述**：
- QClaw 主界面为左右分栏布局（或类似聊天软件布局）
- 左侧为功能导航栏，右侧主区域当前显示二维码绑定页面
- 二维码尺寸较大，方便手机扫描
- 页面底部可能有"使用说明"或"帮助"链接

**步骤 8：手机微信扫码**

1. 打开手机上的微信应用
2. 点击微信右上角的"+"号按钮
3. 在弹出菜单中选择"扫一扫"
4. 将手机摄像头对准电脑屏幕上的二维码
5. 保持适当距离（约 15-25 厘米），确保二维码完整出现在扫描框内
6. 等待手机震动或发出提示音，表示扫描成功

**预期界面描述（手机端）**：
- 微信扫描成功后，手机屏幕显示"QClaw 登录确认"页面
- 页面显示你的微信头像和昵称
- 页面有"确认登录"按钮
- 可能有"取消"按钮

**步骤 9：确认登录**

1. 在手机微信上点击"确认登录"按钮
2. 等待 2-3 秒
3. 电脑端的 QClaw 界面会自动跳转，显示主界面

**预期界面描述（电脑端）**：
- 二维码消失，主界面显示欢迎信息
- 顶部显示你的微信昵称和头像
- 中间区域为对话/任务区域
- 底部有输入框，提示文字类似"输入任务指令..."
- 左侧导航栏可能有"新建任务"、"历史记录"、"设置"等选项

**步骤 10：验证绑定成功**

1. 在 QClaw 底部输入框中输入："你好"
2. 按回车键或点击发送按钮
3. 等待 QClaw 回复

**预期结果**：
- QClaw 回复一条欢迎消息，内容类似"你好！我是 QClaw，你的 AI 助手。有什么可以帮你的吗？"
- 回复速度通常在 3-10 秒内
- 如果收到回复，说明绑定成功，可以正常使用

> **【讲师话术】**
> 到这里，安装和绑定应该都完成了。如果哪一步卡住了，先举手问助教，不要自己乱点。常见问题我等下会集中讲。

### 2.5 基础界面导览

**主界面布局说明**：

1. **顶部栏**：显示当前用户（微信头像+昵称）、网络状态、设置入口
2. **左侧导航栏**：
   - "新建任务"：开始一个新的 Agent 任务
   - "历史记录"：查看之前提交的任务和结果
   - "Skills"：查看可用的技能插件（如搜索、整理、微信推送等）
   - "设置"：配置个人偏好、查看 Token 用量
3. **中间主区域**：显示当前任务的对话流和结果
4. **底部输入框**：输入自然语言指令，按回车发送

**第一次操作建议**：

1. 点击左侧"新建任务"按钮
2. 在底部输入框输入："介绍一下你自己"
3. 按回车发送
4. 观察 QClaw 的回复内容和格式

## 三、实战任务：农林领域 AI 应用案例搜集（15:45 - 17:30，105 分钟）

### 3.1 任务说明

**任务名称**：搜集近 3 年农林领域 AI 应用案例，整理成结构化笔记

**任务背景**：
假设你是某农业大学科研助理，导师让你快速了解 AI 在农林领域的最新应用动态，为课题组下一步研究方向提供参考。你需要搜集 2023-2026 年的相关案例，整理成一份结构清晰的资料。

**任务目标**：
1. 使用 QClaw 完成信息搜集和整理
2. 产出一份包含至少 5 个案例的结构化笔记
3. 笔记需包含：案例名称、应用领域、技术类型、效果数据、来源链接

### 3.2 任务步骤详解

**步骤 1：启动 QClaw 并新建任务**

1. 确保 QClaw 已登录并显示主界面
2. 点击左侧导航栏的"新建任务"按钮
3. 界面中间区域清空，显示"新任务"标题

**步骤 2：输入第一个指令（信息搜集）**

1. 点击底部输入框
2. 输入以下指令（可以复制粘贴，也可以用自己的话表达）：

```
请搜集 2023 年至 2026 年期间，人工智能在农业或林业领域的应用案例。
要求：
1. 至少找到 5 个不同场景的案例
2. 每个案例包含：名称、具体应用领域、使用的 AI 技术、取得的效果或数据、信息来源链接
3. 场景尽量多样化，例如：病虫害识别、作物产量预测、智慧灌溉、林业遥感监测、农产品质量检测等
4. 优先选择有具体数据支撑的案例（如"识别准确率 95%""产量提升 20%"等）
```

3. 按回车键发送指令
4. 等待 QClaw 执行（可能需要 30 秒到 3 分钟，取决于网络速度和任务复杂度）

**预期界面描述**：
- QClaw 显示"正在处理..."或类似状态提示
- 可能分步显示执行过程："正在搜索相关资料..."、"正在整理信息..."
- 最终显示整理好的结果，通常为文本列表或表格形式

**步骤 3：检查结果并补充指令**

1. 阅读 QClaw 返回的结果
2. 检查是否满足以下标准：
   - 案例数量是否 >= 5 个
   - 每个案例是否包含名称、应用领域、技术类型、效果数据、来源链接
   - 场景是否多样化
   - 是否有具体数据支撑

3. 如果不满足，输入补充指令。例如：

```
请再补充 2 个林业领域的案例，重点关注森林火灾监测和木材质量检测方向。
```

或

```
第 3 个案例缺少具体效果数据，请补充准确率或效率提升的百分比。
```

**步骤 4：要求格式化输出**

当内容满意后，输入格式化指令：

```
请将以上所有案例整理成 Markdown 表格格式，表头为：案例名称 | 应用领域 | AI 技术 | 效果数据 | 来源链接
```

**预期结果**：
- QClaw 输出标准 Markdown 表格
- 表格结构清晰，每列对齐

**步骤 5：导出或复制结果**

1. 在 QClaw 界面中找到结果区域
2. 选中全部结果文本（Ctrl+A 全选，或鼠标拖动选择）
3. 复制（Ctrl+C）
4. 打开本地文本编辑器（记事本、VS Code、或 Word）
5. 粘贴（Ctrl+V）保存

**保存建议**：
- 文件名：`Day01-农林AI案例搜集-姓名.md`（计算机专业可用 .md，非计算机专业可用 .txt 或 .docx）
- 保存位置：桌面或专门创建的"AI实训"文件夹

**步骤 6（进阶）：微信推送测试**

如果时间充裕，可以尝试：

```
请将这个表格发送到我微信，以文件形式推送。
```

或

```
请生成一段 200 字的摘要，发到我微信。
```

> **【讲师话术】**
> 这个任务不是考试，没有标准答案。你们搜集的案例可能各不相同，这很正常。重点是体验"给 Agent 下指令 -> 等待结果 -> 检查质量 -> 补充优化"这个完整循环。这是你们未来用 AI 工作的基本流程。

### 3.3 任务评价标准

| 评价维度 | 优秀（90-100） | 良好（75-89） | 合格（60-74） | 需改进（<60） |
|----------|----------------|---------------|---------------|----------------|
| **信息完整性** | 5+ 案例，每个案例 5 要素齐全，有数据支撑 | 5 案例，要素基本齐全，部分有数据 | 4 案例，部分要素缺失 | 案例少于 4 个或要素严重缺失 |
| **结构清晰度** | Markdown 表格规范，分类合理，逻辑清晰 | 表格基本规范，分类较合理 | 有表格但格式混乱，或纯文本罗列 | 无结构，纯复制粘贴 |
| **QClaw 使用熟练度** | 能独立发送指令、检查结果、补充优化、导出保存 | 基本能完成操作，偶尔需要提示 | 需要多次提示才能完成 | 无法独立完成基础操作 |
| **专业相关性** | 案例紧密结合农林领域，场景多样化 | 案例与农林相关，场景有一定多样性 | 部分案例偏离农林主题 | 大部分案例与农林无关 |

### 3.4 参考案例方向（供学生参考）

如果学生不知道搜什么，可以提供以下方向：

1. **病虫害识别**：基于图像识别的作物病虫害自动诊断系统
2. **产量预测**：基于气象数据和机器学习的水稻/小麦产量预测
3. **智慧灌溉**：基于土壤传感器和 AI 的精准灌溉决策系统
4. **林业遥感**：基于卫星遥感和深度学习的森林覆盖变化监测
5. **农产品分级**：基于计算机视觉的水果/蔬菜自动分级分拣
6. **畜牧监测**：基于物联网和 AI 的畜禽健康监测与疫病预警
7. **无人机植保**：AI 路径规划的无人机精准施药

## 四、常见错误排查

### 4.1 安装失败

**现象**：安装程序无法运行，或运行后报错

**排查步骤**：

1. **检查系统版本**：
   - Windows：右键"此电脑" -> 属性，确认是 Windows 10 或 11
   - 如果是 Windows 7，QClaw 可能不支持，需换用网页版或联系助教

2. **检查磁盘空间**：
   - 打开"此电脑"，查看 C 盘可用空间
   - 如果少于 500MB，清理磁盘或更换安装路径到其他分区

3. **关闭杀毒软件**：
   - 部分杀毒软件可能误报 QClaw 安装程序
   - 临时退出 360、腾讯电脑管家等安全软件，重新运行安装

4. **以管理员身份运行**：
   - 右键安装程序 -> "以管理员身份运行"

5. **使用离线安装包**：
   - 如果官网下载的安装包损坏，向助教索取离线安装包

### 4.2 微信绑定问题

**现象**：扫码后无反应，或提示绑定失败

**排查步骤**：

1. **检查网络**：
   - 确认电脑和手机都能正常上网
   - 尝试在电脑浏览器打开 `qq.com`，确认网络通畅

2. **刷新二维码**：
   - 点击二维码下方的"刷新"按钮，生成新二维码
   - 等待 2 秒后重新扫描

3. **检查微信版本**：
   - 手机微信需更新到较新版本（建议 8.0.40 以上）
   - 打开微信 -> "我" -> "设置" -> "关于微信" 查看版本

4. **重新扫码**：
   - 退出微信扫一扫，重新进入
   - 确保二维码完整显示在扫描框内，不要斜着扫

5. **联系助教**：
   - 如果以上步骤都无效，可能是微信账号异常或 QClaw 服务端问题
   - 记录错误提示截图，联系助教处理

### 4.3 网络问题

**现象**：QClaw 提示"网络错误"、"连接超时"、或长时间无响应

**排查步骤**：

1. **检查学校网络限制**：
   - 尝试访问 `qclaw.qq.com`，如果打不开，可能是学校网络对该域名有限制
   - 尝试切换网络：如教室有多个 WiFi，换一个连接；或使用手机热点

2. **检查防火墙**：
   - Windows 防火墙可能阻止 QClaw 联网
   - 前往"控制面板 -> Windows Defender 防火墙 -> 允许应用通过防火墙"，确保 QClaw 被允许

3. **重启 QClaw**：
   - 完全退出 QClaw（右键任务栏图标 -> 退出）
   - 重新启动，等待 10 秒后重试

4. **检查 Token 用量**：
   - 点击 QClaw 设置，查看当日 Token 使用量
   - 虽然每天有 4000 万 Token，但如果异常消耗，可能触发限流

5. **服务端问题**：
   - 如果周围同学都出现同样问题，可能是 QClaw 服务端临时故障
   - 等待 10-15 分钟后重试，或联系助教确认

### 4.4 任务执行问题

**现象**：QClaw 回复"无法完成"、"没有找到相关信息"、或结果质量很差

**排查步骤**：

1. **优化指令表述**：
   - 问题可能是指令太模糊
   - 将"搜一下农业 AI"改为"搜集 2023-2026 年 AI 在病虫害识别领域的应用案例，包含技术类型和效果数据"

2. **分步执行**：
   - 不要一次性要求太多，先让 QClaw 搜案例，再让整理格式
   - 第一步："找 5 个农林 AI 案例"
   - 第二步："把第 3 个案例补充完整"
   - 第三步："整理成表格"

3. **检查 Skills 状态**：
   - 点击左侧"Skills"，确认"搜索"、"整理"等技能已启用
   - 如果 Skills 加载失败，重启 QClaw

4. **降低复杂度**：
   - 如果任务太复杂导致超时，先做一个简化版
   - 例如：先只搜集 3 个案例，不要来源链接，完成后再补充
