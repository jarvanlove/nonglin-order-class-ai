## 五、常见问题速查

### Q1：QClaw 搜不到结果怎么办？

**解决步骤**：
1. 换更宽泛的关键词（如把"水稻良种补贴政策"换成"水稻补贴"）
2. 指定具体网站：`请在农业农村部官网搜索...`
3. 放宽时间范围
4. 如果确实搜不到，可以用老师提供的模拟数据补充

### Q2：JSON 格式报错怎么办？

**解决步骤**：
1. 把文件内容复制到在线 JSON 校验工具（如 json.cn）
2. 看报错位置，检查是否有：
   - 多余的逗号 `,`
   - 引号不匹配 `"`
   - 括号不配对 `{}` `[]`
3. 让组里的计算机同学帮忙看
4. 用 Trae Chat 模式把报错贴给 AI 修复

### Q3：网页显示空白怎么办？

**解决步骤**：
1. 按 `F12` 打开开发者工具，看 Console 里有没有红色报错
2. 检查 JSON 文件路径是否正确
3. 检查 JSON 格式是否正确
4. 用 VS Code 的 Live Server 插件打开（而不是直接双击）

### Q4：字段显示 undefined 怎么办？

**解决步骤**：
1. 检查 JSON 里的字段名是否和代码里的一致
2. 比如代码里用 `publishDate`，JSON 里不能写成 `publish_date`
3. 对照老师发的"数据接口契约"检查字段名

### Q5：Trae 生成的代码报错怎么办？

**解决步骤**：
1. 把报错信息复制下来
2. 打开 Trae Chat 模式
3. 粘贴报错，说：`出现了这个报错，请帮我修复`
4. 如果还是不行，找助教帮忙

### Q6：我们组进度落后了怎么办？

**解决步骤**：
1. 优先保证 P0（核心功能），放弃 P1/P2
2. 组长重新分配任务，让先完成的人帮落后的人
3. 找助教说明情况，助教会帮你们调整任务
4. 记住：完成比完美重要
